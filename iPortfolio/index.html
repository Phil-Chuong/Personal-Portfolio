Gazetteer

A web-based interactive mapping tool designed to enhance geographic exploration through an intuitive and visually engaging interface. Whether you're a traveler, educator, or geography enthusiast, this application provides real-time access to country details, landmarks, and cultural sites. Users can explore layered geographical data, interact with dynamic markers, and access contextual insights seamlessly. By integrating real-time APIs, Gazetteer delivers up-to-date geographical information while offering smooth user interactions through asynchronous data loading and filtering.


Key Technologies:
Frontend: JavaScript, Leaflet.js (interactive maps), Bootstrap (responsive design)
Data Integration: RESTful APIs for real-time geographical data
UX & Interactivity: jQuery, AJAX (asynchronous data fetching), dynamic filtering
Map Enhancements: Leaflet ExtraMarkers, Marker Clustering, EasyButton for UI controls
